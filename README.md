# AllayMc ModInspector

Enterprise-grade mod handshake inspector for Paper servers.

Developer: ElleIsOP

## Features

- Forge/Fabric handshake mod detection
- Blacklist & whitelist system
- SQLite database storage
- Violation scoring
- Discord webhook alerts
- /mods <player> command
- GitHub CI build pipeline

## Requirements

- Paper 1.20+
- ProtocolLib

## Build

mvn clean package

Jar will be located in /target

## Configuration

See config.yml
